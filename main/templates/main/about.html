About This Project

This is a pet project for booking airline tickets. It is designed to demonstrate the functionality of a flight booking system. The project includes the following features:

  • Search for flights based on origin, destination, and departure date.
  • View search results with detailed flight information.
  • Book tickets for flights and enter passenger details.
  • Process payments for ticket bookings.
  • Send email confirmations for booked tickets.
  • User authentication with login and registration functionality.
  • User profile page displaying booked tickets and user information.

This project is built using Django, a powerful web framework for Python, along with Bootstrap 5 for styling and layout.
